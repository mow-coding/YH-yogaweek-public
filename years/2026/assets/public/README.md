# Public Branding Assets

이 폴더는 공개 레포와 GitHub Pages 첫 화면에 쓰는 브랜딩 이미지만 보관합니다.

## Files

- `yeonhui-yoga-week-banner.png`: 공개 페이지 본문 배너입니다.
- `yeonhui-yoga-week-ci.png`: 1:1 비율 CI 이미지입니다.
- `favicon-32.png`: 브라우저 탭용 작은 아이콘입니다.
- `apple-touch-icon.png`: 모바일 홈 화면 저장용 아이콘입니다.
- `site-icon-512.png`: 큰 정사각 아이콘입니다.
- `og-image.png`: 링크 공유 미리보기용 1200x630 이미지입니다.

## Cropping Rule

원본 배너의 오른쪽에는 행사 로고와 핵심 텍스트가 있으므로 오른쪽을 자르면 안 됩니다.

`og-image.png`는 원본 배너를 잘라내지 않고 1200x630 캔버스 안에 맞춰 넣은 뒤, 남는 영역을 원본 배경색에 가까운 주황색으로 채웠습니다. 앞으로 이미지를 다시 만들 때도 오른쪽 로고는 보존하고, 비율 때문에 조정이 필요하면 왼쪽을 줄이거나 배경색 패딩을 늘리는 방식을 우선합니다.
