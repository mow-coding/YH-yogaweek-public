# YH-yogaweek Public

연희 요가 위크 데이터 분석의 공개 공유용 레포지토리입니다.

이 레포지토리는 원본 예약자 정보, 전화번호, API key, 원본 스크린샷, 내부 리뷰 매칭표, Google Drive/Notion/웹 원문을 포함하지 않습니다. 공개 가능한 파생 데이터, 분석 스크립트, 방법론 문서, 통합 보고서만 담습니다.
